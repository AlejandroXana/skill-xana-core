# Documentación para desarrolladores

Material base para construir la documentación técnica de Xana Core dirigida a desarrolladores que trabajen en la plataforma.

**Contenido esperado** (a medida que vaya escribiéndose):
- Estructura de plugins custom (`wom_iCat`, `wom-iCat-Outlet`, `wom-iCat-admin`, `XanaSettings`)
- Hooks y filtros propios de Xana Core (nombre, namespace, cuándo se disparan, qué esperan)
- Esquema de BD (tablas custom, meta keys comunes, user_meta relevantes)
- Flujos de negocio clave (checkout outlet, búsqueda, filtros por catálogo, sistema de permisos por sub-catálogo)
- Cómo añadir un plugin nuevo que integre con la plataforma
- Sistema de multisite en Xana Core (qué sitios comparten qué, cómo se resuelven los menús por contexto)

**Formato**: markdown libre, puede ser un fichero por tema. No fechas en los nombres — es documentación viva que se edita en sitio.
