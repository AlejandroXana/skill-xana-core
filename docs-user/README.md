# Documentación para usuarios finales

Material base para construir manuales de Xana Core dirigidos a los usuarios finales de la plataforma (clientes que gestionan sus catálogos de producto, operadores de outlet, administradores de sub-catálogos, etc.).

**Contenido esperado** (a medida que vaya escribiéndose):
- Guía de uso del back-office (gestión de productos, series, ambientes)
- Flujo de compra en outlet
- Sistema de favoritos
- Preferencias de idioma y multisite
- Exportación/importación de datos

**Formato**: markdown libre, orientado a persona no técnica. Screenshots (como referencias) bienvenidos cuando ayuden.

**Nota**: la redacción final pulida para publicar puede requerir otro tono que el resto de la skill (más pedagógico, menos jerga). Estos ficheros son el *source of truth* del contenido; el estilo definitivo se adapta al canal (web, PDF, help-desk).
